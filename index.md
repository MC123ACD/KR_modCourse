# 说明
本文档使用伪代码类型注解：

| 类型注解       | <center>说明</center>          |
| ---------- | ---------------------------- |
| `bool`     | 布尔值                          |
| `float`    | 小数（浮点）                       |
| `int`      | 整数（整形）                       |
| `num`      | 浮点与整形                        |
| `str`      | 字符串                          |
| `table`    | 表                            |
| `list`     | 纯数组的表                        |
| `dict`     | 纯字典的表                        |
| `func`     | 函数                           |
| `thread`   | 协程                           |
| `userdata` | C 数据结构                       |
| `any`      | 以上所有类型的集合                    |
| `vec2`     | 二维向量 `table{x: num， y: num}` |
| `...`      | 不定参数                         |
| `->`       | 返回值，返回 `nil` 时省略             |
| \|         | 或                            |
| `?`        | 选填项                          |
