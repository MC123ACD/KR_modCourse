### 示例
- **问题**：Lua 我想检查表里面有没有元素应该怎么写
- **AI**：
	1. 使用 `next` 函数 `if not next(表) then`
	2. `if #表 == 0 then`（只对数组部分有效）

- **问题**：我试了第一种，现在想检查表格是否有某个键，怎么写
- **AI**：
	1. 使用 `if 表.键 then`
	2. 或者使用 `if 表["键"] then`

# 类型注解
本文档使用伪代码类型注解：

| 类型注解    | <center>说明</center> | 类型注解       | <center>说明</center>                        |
| ------- | ------------------- | ---------- | ------------------------------------------ |
| `bool`  | 布尔值                 | `thread`   | 协程                                         |
| `float` | 小数（浮点）              | `userdata` | C 数据结构                                     |
| `int`   | 整数（整形）              | `any`      | 以上所有类型                                     |
| `num`   | 浮点与整形               | `vec2`     | 二维向量 `table{x， y}`                         |
| `str`   | 字符串                 | `rect`     | 矩形 `dict{pos = 位置: vec2, size = 大小: vec2}` |
| `table` | 表                   | `...`      | 不定参数                                       |
| `list`  | 纯数组的表               | `->`       | 返回值，返回 `nil` 时省略                           |
| `dict`  | 纯字典的表               | \|         | 或                                          |
| `func`  | 函数                  | `?`        | 选填项                                        |
