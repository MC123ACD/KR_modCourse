5. 字符串

5.1 字符串基础

字符串是编程中最常用的数据类型之一。在 Lua 中，字符串是不可变的（你不能修改其中的某个字符，只能创建新字符串），但 Lua 提供了丰富的函数来操作它们。

5.1.1 字符串的创建

Lua 提供了多种方式创建字符串：

-- 1. 双引号
local str1 = "Hello, Lua!"

-- 2. 单引号
local str2 = 'Hello, World!'

-- 3. 长括号（多行字符串，可以包含换行和引号，无需转义）
local str3 = [[
这是多行字符串
可以包含换行和"各种引号"
而无需转义]]

-- 4. 带等号的长括号（用于嵌套，等号数量可任意）
local str4 = [=[
多层嵌套 [[不会结束字符串]]
]=]

5.1.2 转义符

在普通引号字符串中，可以使用转义符表示特殊字符：

-- 转义字符，用于输入特殊字符
local escaped = "第一行\n换行\t制表符（缩进）\"引号\\反斜杠"
print(escaped)
--[[ 输出
第一行
换行 制表符（缩进）"引号\反斜杠
]]

常用转义符

转义符 描述
\n 换行
\t 制符表（缩进）
\\ 反斜杠
\' 单引号
\" 双引号

5.1.3 字符串是不可变的

当你“修改”字符串时，其实是创建了一个新字符串，原字符串不变。

local a = "Hello"
local b = a       -- b 和 a 指向同一个字符串
a = a .. " World" -- 这创建了一个新字符串，原字符串不变

print(b)	--> Hello（原字符串没变）
print(a)	--> Hello World（a 现在指向新字符串）

5.2 字符串基本操作

5.2.1 # - 字符串长度

返回字符串的字节数（注意：一个中文字符通常占 3 个字节，取决于编码）。

local str = "Hello, Lua!"
print(#str)            --> 11（字节数）
警告

中文字符在 UTF-8 编码下通常占 3 个字节，所以 # 返回的不是“字符数”，而是字节数。例如 "你好" 的长度是 6，而不是 2。

5.2.2 table.concat - 字符串拼接

当你需要把很多小字符串拼成一个长字符串时，直接用 .. 连接会很慢。更好的办法是先把它们放进一个表，再用 table.concat 一次性拼接。

table.concat(表, 连接字符?, 起始索引?, 结束索引?) -> 拼接后的字符串

示例：

local words = {"Hello", "World", "from", "Lua"}
local sentence = table.concat(words, " ")  -- 用空格连接
print(sentence)  --> Hello World from Lua

5.2.3 string.lower - 大写转小写

string.lower(字符串) -> 小写字符串

示例：

print(string.lower("AAAbbbCCC"))	--> aaabbbccc

5.2.4 string.upper - 小写转大写

string.upper(字符串) -> 大写字符串

示例：

print(string.lower("AAAbbbCCC"))	--> AAABBBCCC

5.2.5 string.rep - 字符串重复

string.rep(重复的字符串, 重复次数) -> 字符串

示例：

-- 生成分隔线
print(string.rep("-", 30))  --> ------------------------------

-- 进度条
local function progress_bar(percent, width)
    local filled = math.floor(percent * width / 100)
    return "[" .. string.rep("=", filled) .. string.rep(" ", width - filled) .. "]"
end

print(progress_bar(50, 20))  --> [==========          ]

5.2.6 string.find - 查找子串

返回子串的起始和结束位置，找不到返回 nil。

string.find(字符串, 子串, 搜索位置?, 是否不使用正则模式?) -> 子串起始位置, 子串结束位置
  • 搜索位置可以是一个负数，表示从后往前数的字符个数。

示例：

local text = "Hello, Lua programming!"
local start, finish = string.find(text, "Lua")
print(start, finish)  --> 8, 10（第 8 到第 10 个字符）

-- 从指定位置开始查找
local pos = string.find(text, "o", 6)  -- 从第 6 个字符开始找 "o"

5.2.7 string.sub - 截取子串

string.sub(字符串, 截取起始位置, 截取结束位置?) -> 截取的子串
  • 截取结束位置默认为 -1（截取到最后一个）

示例：

local str = "The quick brown fox"

print(string.sub(str, 1, 3))   --> The（第 1 到第 3 个字符）
print(string.sub(str, -3))     --> fox（最后 3 个字符）
print(string.sub(str, 5, 9))   --> quick

5.3 - 模式匹配（轻量级正则）

模式匹配（Pattern Matching） 正是为了解决“模糊但规律”的字符串处理问题而生的工具。Lua 提供了轻量级的模式匹配系统（不是完整的正则表达式，但足够强大）

常用模式：

符号 含义 示例
. 任意字符 ("abc123", ".") → "a"
%a 字母 ("abc123", "%a") → "a"
%d 数字 ("abc123", "%d") → "1"
%w 字母或数字 ("123abc!", "%w") → "1"
%s 空白字符（空格、换行等） (" ", "%s") → ""
%l 小写字母 ("abc123", "%l") → "a"
%u 大写字母 ("ABC123", "%u") → "A"
%p 标点符号 (",./", "%s") → ","
%% 百分号 ("100%", "%%") → "%"

5.3.1 string.match - 匹配单个模式

string.match(字符串, 匹配模式, 起始位置?) -> 匹配的子串

示例：

-- 匹配第一个数字
print(string.match("abc123!@#", "%d"))  --> 1

-- 匹配第一个字母
print(string.match("abc123!@#", "%a"))  --> a

5.3.2 string.gmatch - 遍历所有匹配

string.gmatch(字符串, 匹配模式) -> 迭代器函数

示例：

-- 找出所有数字
for digit in string.gmatch("a1b2c3", "%d") do
    print(digit)  --> 1 2 3
end

5.3.3 字符集 [] - 匹配集合内字符

示例：

local text = "abc123!@#DEF"

-- 匹配数字
for match in string.gmatch(text, "[123]") do
    print(match)  --> 1 2 3
end

-- 匹配小写字母或数字
for match in string.gmatch(text, "[abc123]") do
    print(match)  --> a b c 1 2 3
end

5.3.4 重复修饰符 - 控制匹配次数

符号 含义 示例
* 0 次或多次（贪婪） "a*" 匹配 "aaa" 或 ""
+ 1 次或多次 "a+" 匹配 "aaa"
? 0 次或 1 次 "a?" 匹配 "a" 或 ""
- 0 次或多次（非贪婪） 尽量少匹配

示例：

local text = "aaabbcccddddeeee"

-- + 匹配连续相同的字母
for match in string.gmatch(text, "a+") do
    print(match)  --> aaa
end

5.3.5 捕获 () - 提取部分内容

括号内的部分会被单独捕获并返回。

local date = "2024-12-27"
local year, month, day = string.match(date, "(%d+)-(%d+)-(%d+)")
print(year, month, day)  --> 2024 12 27

-- 解析键值对
for name, age in string.gmatch("alice:18,bob:22", "(%w+):(%d+)") do
    print(name, age)  --> alice 18 bob 22
end

5.3.6 string.gsub - 替换

将匹配的部分替换为新内容，返回新字符串和替换次数。

string.gsub(字符串, 匹配模式, 替换为的字符串, 替换次数?) -> 新字符串, 替换次数
  • 不指定替换次数将会替换所有匹配的字符串

示例：

-- 基本替换
local text = "Hello World"
local replaced = string.gsub(text, "World", "Lua")
print("基本替换: " .. replaced)	--> 基本替换: Hello Lua

-- 限制替换次数
local text = "Hello World World World"
local replaced = string.gsub(text, "World", "Lua", 2)
print("替换两次: " .. replaced)	--> 替换两次: Hello Lua Lua World

-- 使用模式匹配替换
local text = "Price: $10.99, Discount: 20%"
local replaced = string.gsub(text, "%$%d+%.%d%d", "[价格隐藏]")
print("隐藏价格: " .. replaced)	--> 隐藏价格: Price: [价格隐藏], Discount: 20%

5.4 string.format - 字符串格式化

我们经常需要将变量值嵌入到字符串中。最直观的做法是使用字符串拼接运算符  .. 。

这种写法虽然可行，但当字符串变得复杂、变量增多或需要对输出格式做精细控制时，拼接方式就会暴露出诸多问题。字符串格式化正是为了解决这些问题而出现的——它通过占位符来定义字符串模板，然后将实际值按指定规则替换进去，同时支持对齐、精度、进制转换等额外处理。

string.format(被格式化的字符串, 替换值...) -> 格式化字符串

常用格式符：

格式符 含义 示例
%s 字符串 string.format("%s", "hello") → "hello"
%d 整数 string.format("%d", 12.34) → "12"
%f 浮点数 string.format("%f", 12.34) → "12.340000"
%.2f 保留两位小数的浮点数 string.format("%.2f", 12.34) → "12.34"
%e 科学计数法 string.format("%e", 1234) → "1.234000e+03"

示例：

local name = "小明"
local age = 18
local formatted = string.format("姓名: %s, 年龄: %d", name, age)
print(formatted)	--> 姓名: 小明, 年龄: 18

对齐和填充

local price = 19.99

-- 宽度10，右对齐
print(string.format("%10.2f", price))   -->      19.99

-- 宽度10，左对齐（加负号）
print(string.format("%-10.2f", price))  --> 19.99    

-- 前导零（宽度4）
print(string.format("%04d", 19))        --> 0019

5.5 字符串与数字转换

5.5.1 tostring - 转字符串

tostring(值) -> 字符串

示例：

local num = 123.456
local str_num = tostring(123.456)

print(string.format("  数字: %.3f，类型： %s", num, type(num)))
print(string.format("  字符串: \"%s\"，类型：%s", str_num, type(str_num)))
--[[ 输出
数字: 123.456，类型：number
字符串: "123.456"，类型：string
]]
信息

连接或格式化时自动会将数字转化为字符串：

print("连接：" .. num)    --> 连接：123.456
local fmt = string.format("格式化：%f", num)
print(fmt)    --> 格式化：123.456000

5.5.2 tonumber - 转数字

tonumber(要转化的字符串) -> 数字

示例：

local str = "123.456"
local num = tonumber(str)

print(string.format("  字符串: \"%s\"，类型：%s", str, type(str)))
print(string.format("  数字: %.3f，类型： %s", num, type(num)))
--[[ 输出
  字符串: "123.456"，类型：string
  数字: 123.456，类型： number
--]]

5.6 VScode 搜索与替换

5.6.1 搜索

VScode 中搜索可以使用 ctrl + F 快捷键或在搜索选项中搜索。

搜索的三种模式：

  1. 区分大小写
  2. 全字匹配：完全匹配字符，多一个字少一个字都不行，搜索 lua 只能匹配到 lua，alua，luaa 都无法匹配
  3. 正则表达式：h+ 匹配 hhh...，(.+) 匹配所有括号与括号内的内容 (abc...)

5.6.2 替换

点击搜索框左侧的箭头即可打开替换模式，搜索的内容将会被替换，可以全部替换也可以在搜索结果上替换单个。

5.6.3 高级替换

由于 VScode 替换功能较少只有保留大小写，可以使用 Find and Transform 插件编写 JavaScript 替换脚本。

Find and Transform 可以做到：

  • 使用 JavaScript 脚本进行复杂的文本处理
  • 在替换中使用正则表达式捕获组
  • 执行数学计算和字符串操作
  • 应用条件逻辑和循环

1. 配置键盘快捷键

按 Ctrl+Shift+P 快捷键打开命令面板，输入 Preferences: Open Keyboard Shortcuts (JSON) 打开键盘快捷键配置文件。

配置模板：

[
	{
		"key": "alt+1",        		// 快捷键
		"command": "findInCurrentFile",
		"args": {
			"find": "匹配模式",   		// 可以是纯文本或正则表达式
			"replace": "替换内容", 		// 可以是纯文本或JavaScript代码
			"isRegex": true,     		// 是否使用正则表达式
			"matchCase": false,  		// 是否区分大小写（可选）
			"restrictFind": "document"	// 搜索范围（可选）
    	}
	}
]

2. 基础语法

正则表达式捕获组
  • (pattern) - 创建捕获组
  • $1, $2, $3 - 引用捕获组
  • \\d+ - 匹配数字（需要双反斜杠转义）
JavaScript 替换语法
"replace": [
	"$${",                // 开始JavaScript代码
	"// 你的代码",
	"return '结果';",      // 必须返回字符串
	"}$$"                 // 结束
]

3. 实用示例

示例 1：条件替换数字

需求：将 hello lua4 到 hello lua6 中的数字乘以 11，其他保持不变。

print("hello lua1")
print("hello lua2")
print("hello lua3")
print("hello lua4")
print("hello lua5")
print("hello lua6")

[
	{
		"key": "alt+1",
		"command": "findInCurrentFile",
		"args": {
			"find": "(hello lua)(\\d+)",
			"replace": [
				"$${",
				"// 检查数字是否在4-6范围内",
				"if ($2 >= 4 && $2 <= 6) {",
				"  // 乘以11并返回",
				"  const newNum = parseInt($2) * 11;",
				"  return $1 + newNum;",
				"} else {",
				"  // 保持原样",
				"  return $1 + $2;",
				"}",
				"}$$"
			],
		"isRegex": true
		}
	}
]

执行结果：

-- 保存 json 后打开你的代码文件按快捷键 alt + 1 替换。
print("hello lua1")
print("hello lua2")
print("hello lua3")
print("hello lua44")
print("hello lua55")
print("hello lua66")
示例 2：格式化文本

需求：将 hello luaX 格式化为 hello lua, num is X

[
	{
		"key": "alt+2",
		"command": "findInCurrentFile",
		"args": {
			"find": "(hello lua)(\\d+)",
			"replace": "$1, num is $2",
			"isRegex": true
		}
	}
]

执行结果：

print("hello lua, num is 1")
print("hello lua, num is 2")
print("hello lua, num is 3")
print("hello lua, num is 44")
print("hello lua, num is 55")
print("hello lua, num is 66")

5.7 练习

练习 1：解析日志

[10:20:30]ERROR.take_damage: 无法造成伤害，因为不是一个有效目标

编写函数，使用模式匹配提取以下信息，返回一个表，表中存储一个函数可以打印格式化后的日志信息：

  • time：时间部分
  • level：日志级别
  • func：函数名
  • message：报错信息

示例：

local parsed = parse_log("[10:20:30]ERROR.take_damage: 无法造成伤害，因为不是一个有效目标")
parsed:print()

--[[ 输出
时间：10:20:30
级别：ERROR
函数：take_damage
信息：无法造成伤害，因为不是一个有效目标
]]

练习 2：文本分析器

编写一个文本分析器程序，可以：

  1. 分析给定的字符串有几行文本。
  2. 分析给定的字符串有多少个单词。
  3. 分析给定的字符串有总字符数。
  4. 统计并打印出现最频繁的单词。
    • 根据出现次数排序。
    • 仅显示前 5 个单词。
    • 使用字符串格式化对齐。

示例：

local test_text = [[
Lua is a powerful, efficient, lightweight, embeddable scripting language.
It supports procedural programming, object-oriented programming, functional
programming, data-driven programming, and data description.

Lua combines simple procedural syntax with powerful data description
constructs based on associative arrays and extensible semantics.
Lua is dynamically typed, runs by interpreting bytecode with a
register-based virtual machine, and has automatic memory management with
incremental garbage collection, making it ideal for configuration,
scripting, and rapid prototyping.
]]

analysis:analyze_text(test_text)

local data = analysis.data

print("基本统计:")
print("  总字符数: " .. data.total_chars)
print("  行数: " .. data.lines)
print("  单词数: " .. data.word_count)

print("\n 最频繁的单词:")
for i, item in ipairs(data.most_frequent) do
    print(string.format("  %2d. %-15s: %d 次", i, item.word, item.count))
end
--[[ 输出
基本统计:
  总字符数: 582
  行数: 11
  单词数: 70

 最频繁的单词:
   1. and            : 4 次
   2. programming    : 4 次
   3. lua            : 3 次
   4. with           : 3 次
   5. a              : 2 次
]]
