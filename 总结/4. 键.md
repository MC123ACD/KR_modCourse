# 精灵

| <center>键</center>       | <center>作用</center> | <center>键</center> | <center>作用</center> |
| ------------------------ | ------------------- | ------------------ | ------------------- |
| `animated`               | 精灵是否启用动画            | `exo`              | 是否为骨骼动画             |
| `group`                  | 动画组                 | `exo_hide_prefix`  | 骨骼动画是否隐藏前缀          |
| `prefix`                 | 动画前缀                | `z`                | 图层                  |
| `name`                   | 动画名称，禁用动画则可以指定图像名称  | `draw_order`       | 绘制顺序                |
| `anchor`                 | 锚点                  | `size_names`       | 尺寸变体名称              |
| `offset`                 | 位置偏移量               | `size_scales`      | 尺寸缩放比例              |
| `pos`                    | 位置坐标                | `sort_y`           | 启用 Y 轴排序            |
| `loop`                   | 是否循环播放动画            | `sort_y_offset`    | Y 轴排序偏移量            |
| `loop_forced`            | 强制循环行为              | `sync_idx`         | 同步索引                |
| `flip_x`                 | 水平翻转                | `time_offset`      | 动画时间偏移              |
| `r`                      | 旋转角度                | `hide_after_runs`  | 播放 N 次后隐藏           |
| `scale`                  | 缩放比例                | `fps`              | 帧率(每秒帧数)            |
| `angles`                 | 角度设置                | `random_ts`        | 随机开始时间              |
| `angles_flip_horizontal` | 水平翻转角度              | `ignore_start`     | 忽略起始帧               |
| `angles_flip_vertical`   | 垂直翻转角度              | `ts`               | 当前时间戳               |
| `angles_custom`          | 自定义角度               | `runs`             | 已完成播放次数             |
| `angles_stickiness`      | 角度粘滞性               | `frame_idx`        | 当前帧索引               |
| `alpha`                  | 透明度(0-255)          | `frame_name`       | 当前帧名称               |
| `hidden`                 | 精灵是否隐藏              | `sync_flag`        | 同步标志                |
