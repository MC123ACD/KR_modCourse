总结用于存放一些表格、函数说明等

函数说明使用伪代码类型注解

| 类型注解    | 说明                             |
| ------- | ------------------------------ |
| `float` | 小数                             |
| `int`   | 整数                             |
| `str`   | 字符串                            |
| `table` | 表，后面可能会有表内元素类型注解               |
| `func`  | 函数                             |
| `vec2`  | 二维向量 `table{ x: num， y: num }` |
| `num`   | 小数 `float` 和整数 `int`           |
| `...`   | 不定参数                           |
| `->`    | 返回值，返回 `nil` 时省略               |

# 名词说明
### [一] 插入、更新、移除系统
插入、更新、移除系统特指 `all/systems` 内的所有对应函数

若插入系统与移除系统其中任意一个函数返回假将会中断后续其他函数
