### [一] 获取所有键
```lua
table.keys(表: table) -> 包含表中所有键的列表: list
```

### [二] 根据键值获取键
```lua
table.keyforobject(表: table, 键值: any) -> 键: any
```

### [三] 检查是否包含指定元素
```lua
table.contains(表: table, 键值: any) -> 是否包含键值: bool
```

### [四] 浅拷贝表
```lua
table.clone(表: table) -> 浅拷贝的表: table
```

### [五] 深拷贝表
```lua
table.deepclone(表: table) -> 深拷贝的表: table
```

### [六] 浅合并表
```lua
table.merge(表1: table, 表2: table, 是否创建新表合并: bool) -> 浅合并后的表: table
```
- 如果不创建新表合并将会修改表 1

### [七] 深合并表 
```lua
table.deepmerge(表1: table, 表2: table, 是否合并到新表?: bool) -> 深合并后的表: table
```
- 如果不创建新表合并将会修改表 1

### [八] 列表合并
```lua
table.append(表1: table, 表2: table, 是否合并到新表?: bool) -> 合并后的列表: list
```

### [九] 反转列表
```lua
table.reverse(表: table, 是否深度反转?: bool) -> 反转后的列表: list
```

### [十] 统计元素数量
```lua
table.count(表: table, 过滤函数?: func) -> 统计到的数量: int
```
- 为过滤函数传递迭代到的：`键: any, 键值: any`

### [十一] 查找指定元素
```lua
table.find(表: table, 元素键值: any|过滤函数: func) -> 键: any
```
- 为过滤函数传递迭代到的：`键: any, 键值: any`

### [十二] 过滤元素
```lua
table.filter(表: table, 过滤函数: func) -> 过滤后的列表: list
```
- 为过滤函数传递迭代到的：`键: any, 键值: any`

### [十三] 映射转换
```lua
table.map(表: table, 映射函数: func) -> 转换后表: table
```
- 为映射函数传递迭代到的：`键: any, 键值: any`
- 映射函数返回值：`键: any, 键值: any`
- 映射函数返回单个值将会被序列化
- 映射函数返回两个值将会保留键值映射

### [十四] 归约元素
```lua
table.reduce(表: table, 归约函数: func) -> 结果: any
```
- 为归约函数传递迭代到的：`键值: any`
- 示例：
```lua
-- 求和
local numbers = { 10, 20, 30, 40 }

local sum = table.reduce(numbers, function(element, accumulator)
    return element + accumulator
end)

print("求和结果:", sum)  -- 100
```

### [十五] 获取最大元素
```lua
table.maxv(表: table) -> 键: any, 键值: any
```

### [十六] 获取最小元素
```lua
table.minv(表: table) -> 键: any, 键值: any
```

### [十七] 列表切片
```lua
table.slice(表: table, 起始索引: int, 结束索引: int) -> 结果: list
```

### [十八] 根据键值移除元素
```lua
table.removeobject(表: table, 键值: any)
```

### [十九] 随机获取列表单个元素
```lua
table.random(表: table) -> 元素的值: any, 元素索引: int
```

### [二十] 随机获取列表元素
```lua
table.random_order(表: table) -> 获取到的元素列表: list
```
