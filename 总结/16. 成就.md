
### 1. 从存档加载成就数据
```lua
A:load() -> 成就数据: table
```

### 2. 保存成就数据到存档
```lua
A:save()
```

### 3. 获取成就配置数据
```lua
A:get_data(成就名: str) -> 成就数据: table
```

### 4. 获取成就当前计数值
```lua
A:get_count(成就名: str) -> 计数值: int
```

### 5. 检查是否已获得成就
```lua
A:have(成就名: str) -> 是否已获得: bool
```

### 6. 获得指定成就
```lua
A:got(成就名: str)
```

### 7. 设置成就的标志位
```lua
A:flag(成就名: str, 标志值: int) -> 当前标志值: int, 成就数据: table
```

### 8. 设置标志位检查并获得成就
```lua
A:flag_check(成就名: str, 标志值: int) -> 当前标志值: int, 成就数据: table
```

### 9. 增加成就计数值
```lua
A:inc(成就名: str, 增加值: int) -> 当前计数值: int, 成就数据: table
```

### 10. 增加计数值检查并获得成就
```lua
A:inc_check(成就名: str, 增加值: int) -> 当前计数值: int, 成就数据: table
```

### 11. 通用检查并获得成就
```lua
A:ge_check(成就名: str, 当前值: int) -> 当前值: int, 成就数据: table
```

### 12. 记录最大值
```lua
A:high(成就名: str, 新值: int) -> 当前最大值: int, 成就数据: table
```
- 仅当新值更大时更新

### 13. 记录最大值检查并获得成就
```lua
A:high_check(成就名: str, 新值: int) -> 当前最大值: int, 成就数据: table
```

### 14. 开始计时
```lua
A:lap_start(成就名: str, 时间戳: num) -> 起始时间: num, 成就数据: table
```
- 记录起始时间戳

### 15. 结束计时并检查并获得成就
```lua
A:lap_check(成就名: str, 结束时间戳: num) -> 结束时间: num, 成就数据: table
```

### 16. 统计活动中的特定类型状态效果数量
```lua
A:count_active_mods(模板名: str, 过滤器: func) -> 数量: int
```

### 17. 重置指定组的计数器
```lua
A:reset_counters(组名: str) -> nil
```
