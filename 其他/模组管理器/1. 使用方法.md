# 创建 mod 目录
##### 1. 在 mods 目录下创建你的模组目录
![[Pasted image 20251124202637.png|525]]
名称请取一个高辨识度并且尽量避免与其他模组重名

##### 2. 复制模板
将 `mods/baisic_mod_template` 的文件全部复制到你的模组目录，并将 `baisic_mod_template` 文件重命名为你的模组名称（与目录相同）
![[Pasted image 20251124205340.png|925]]
##### 3. 填写配置文件
打开你的模组目录下的 `config` 文件，并根据描述填写

# 修改模板
##### 1. 打开 `game_templates` 与你的模组模块，用于对比

##### 2. 在 `game_templates` 查找要修改的键
示例：
假如要修改熊人的最大血量（`hp_max`）

game_templates：
```lua
tt = RT("hero_ignus", "hero")
AC(tt, "melee", "timed_attacks")
tt.hero.level_stats.armor = {...}
tt.hero.level_stats.hp_max = {
	400,
	430,
	460,
	490,
	520,
	550,
	580,
	610,
	640,
	670
}
...
```

你的模组模块：
```lua
function hook.E.load(load, self)
	load(self)
	
	-- tt 改为 T("hero_ignus")
	T("hero_ignus").hero.level_stats.hp_max = {
		100,
		200,
		300,
		400,
		500,
		600,
		700,
		800,
		900,
		1000
	}
end
```
