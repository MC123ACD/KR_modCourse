# 第 14 章：Love 2D

### 14.1 Love 2D 简介

Love 2D 是一个免费的开源 2D 游戏引擎，使用 Lua 作为脚本语言。

**Love 2D 主要特性**:
1. 简单易用的 API
2. 跨平台支持（Windows, macOS, Linux, Android, iOS）
3. 内置物理引擎（Box2D）
4. 支持多种图像、音频格式
5. 强大的着色器支持
6. 网络功能
7. 文件系统访问
