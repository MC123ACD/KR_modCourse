
## 第一步：准备核心工具

| 工具                        | 作用             | 获取方式                                                                    |
| ------------------------- | -------------- | ----------------------------------------------------------------------- |
| VS Code                   | 代码编辑器          | [官网下载](https://code.visualstudio.com/)                                  |
| LuaForWindows             | Lua 运行环境       | [蓝奏云网盘下载](https://wwuk.lanzouo.com/b007u3i2jg)  <br>密码:fv51             |
| 整合包 (VScode_KR_workspace) | 预配置的工作区+Love2D |                                                                         |
| Python                    | 运行脚本           | [微软商店安装](https://apps.microsoft.com/detail/9nq7512cxl7t?hl=zh-CN&gl=CN) |

## 第二步：放置整合包

将解压后的 `VScode_KR_workspace` 文件夹放到 **D 盘根目录**或任意**全英文路径**（例如 `C:/dev/workspace`）。
![[Pasted image 20250706200745.png|361]]

## 第三步：安装环境组件

1. **双击打开工作区文件**  
进入整合包根目录，双击 `VScode_KR_workspace.code-workspace`。
![[Pasted image 20251206205112.png|438]]

2. **安装推荐扩展**  
VS Code 右下角会弹出提示 → 点击「安装所有推荐的扩展」。  
![[Pasted image 20250710092722.png|404]]

3. **安装 Lua 并重启**  
安装网盘中的 `LuaForWindows.exe`，同意条款一路「Next」完成安装。
![[Pasted image 20251206202705.png|408]]

4. **安装 Python**
打开 Python Install Manager 根据提示输入 y 完成安装。
![[Pasted image 20260413153552.png|606]]

5. **重启电脑**
重启电脑使环境变量生效。

## 第四步：导入游戏项目

1. 将游戏本体 `.exe` 解压。
2. 将解压后里面的文件放入整合包根目录下的 `src` 文件夹中。  
3. 遇到文件重复提示时，选择**不替换**。  

![[Pasted image 20260413141043.png|883]]
![[Pasted image 20260413141135.png|322]]

## 第五步：配置游戏项目
运行 `配置游戏项目.py` 脚本配置游戏项目即可。
![[Pasted image 20260413162718.png]]
