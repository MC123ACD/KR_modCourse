
本文档使用伪代码类型注解：

| 类型注解    | <center>说明</center> | 类型注解       | <center>说明</center>                        |
| ------- | ------------------- | ---------- | ------------------------------------------ |
| `bool`  | 布尔值                 | `thread`   | 协程                                         |
| `float` | 小数（浮点）              | `userdata` | C 数据结构                                     |
| `int`   | 整数（整形）              | `any`      | 以上所有类型                                     |
| `num`   | 浮点与整形               | `vec2`     | 二维向量 `table{x， y}`                         |
| `str`   | 字符串                 | `rect`     | 矩形 `dict{pos = 位置: vec2, size = 大小: vec2}` |
| `table` | 表                   | `...`      | 不定参数                                       |
| `list`  | 纯数组的表               | `->`       | 返回值，返回 `nil` 时省略                           |
| `dict`  | 纯字典的表               | \|         | 或                                          |
| `func`  | 函数                  | `?`        | 选填项                                        |
| `class` | 类（实际还是表）            |            |                                            |
