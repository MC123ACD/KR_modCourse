
## 8.1 什么是元表？

元表（**Metatable**）是 Lua 中一种特殊的表，它可以**定义另一个表的行为**。你可以把元表理解为一张“规则清单”——它规定了当你对一个表做某种操作时（比如加法、索引、比较），应该执行什么动作。

### 8.1.1 为什么需要元表？

- **运算符重载**：让表支持 `+`、`-`、`*` 等运算
- **自定义行为**：改变表读取、赋值时的默认逻辑
- **面向对象**：实现继承、私有属性等特性
- **保护数据**：创建只读表、带类型检查的表

#### 没有元表时的限制

```lua
local vector1 = {x = 10, y = 20}
local vector2 = {x = 5, y = 15}

-- 尝试相加两个表 → 报错！
local sum = vector1 + vector2  -- 错误！table 不支持加法

-- 比较两个表 → 比较的是内存地址，不是内容
print(vector1 == vector2)  --> false（即使值相同）

-- 默认打印 → 只显示内存地址
print(vector1)  --> table: 0x000001（不友好）
```

元表就是为了解决这些问题而生的。

## 8.2 元方法的类型

元表里可以定义多个特殊函数，每个函数的名字以两个下划线开头，比如 `__add`、`__index`。这些函数称为**元方法**。

| 分类     | 元方法                                                           | 对应操作                                 |
| ------ | ------------------------------------------------------------- | ------------------------------------ |
| **算术** | `__add`, `__sub`, `__mul`, `__div`, `__mod`, `__pow`, `__unm` | `+`, `-`, `*`, `/`, `%`, `^`, 负号 `-` |
| **关系** | `__eq`, `__lt`, `__le`                                        | `==`, `<`, `<=`                      |
| **库**  | `__tostring`, `__concat`                                      | `print()`、`..` 连接                    |
| **索引** | `__index`, `__newindex`                                       | 索引不存在的键、给不存在的键赋值                     |
| **调用** | `__call`                                                      | 把表当作函数调用 `table()`                   |

**示例**：向量类

```lua
local Vector = {}

-- 创建新向量
function Vector.new(x, y)
    local v = {x = x, y = y}
    setmetatable(v, Vector)   -- 把 Vector 表作为 v 的元表
    
    return v
end

-- 定义加法
function Vector.__add(a, b)
	-- 每次创建新向量，避免引用类型的问题
    return Vector.new(a.x + b.x, a.y + b.y)
end

-- 定义打印格式
function Vector.__tostring(v)
    return string.format("Vector(%d, %d)", v.x, v.y)
end

-- 使用
local v1 = Vector.new(10, 20)
local v2 = Vector.new(5, 15)

print(v1)   	--> Vector(10, 20)
print(v2)   	--> Vector(5, 15)
print(v1 + v2)	--> Vector(15, 35)
```

> [!Info]+ 信息
> - 没有大于操作的元方法，进行 `>`, `>=` 操作时会调用反转操作数后的 `__lt`, `__le`。
> - 没有每次索引表（无论是否存在）的元方法，但可以通过中间人（代理表）来实现：
> ```lua
> local function create_table()
>     return setmetatable({}, {
>         __index = function(table, key)
>             print("索引: " .. key))
>         end
>     })
> end
> 
> local new_table = create_table()
> -- 实际每次在操作 proxy 空表
> a = new_table.a	--> 索引: a
> b = new_table.b	--> 索引: b
> ```

## 8.3 元表基础函数

### 8.2.1 setmetatable - 给表绑定元表
```lua
setmetatable(表: table, 元表: table) -> 设置元表后的表: table
```

**示例**：
```lua
local mytable = {}

local metatable = {
    __add = function()
        return "add"
    end
}

-- 把 metatable 绑定到 mytable 上
setmetatable(mytable, metatable)

print(mytable + 1)		--> add
```

### 8.2.2 getmetatable - 获取元表
```lua
getmetatable(表: table) -> 元表: table
```

**示例**：
```lua
local mytable = {}
local metatable = {}

setmetatable(mytable, metatable)

local mt = getmetatable(mytable)
print(mt == metatable)  --> true
```

### 8.2.3 rawget - 绕过元方法索引
```lua
rawget(表: table, 索引: any) -> 值: any
```

**示例**：
```lua
local danger_table = setmetatable({}, {
    __index = function(table, key)
        return table[key]
    end
})

-- print(danger_table.v)	-- 无限递归栈会溢出

-- 使用 rawget 绕过元方法索引，避免栈溢出
print(rawget(danger_table, "v"))  --> nil
```

### 8.2.4 rawset - 绕过元方法赋值
```lua
rawset(表: table, 索引: any, 值: any)
```

**示例**：
```lua
local secure_table = setmetatable({}, {
    __newindex = function(table, key, value)
        if key == "ban_key" then
            print(string.format("无法设置键: %s!", key))
        end
    end
})

secure_table.ban_key = "new"    --> 无法设置键: ban_key!
print(secure_table.ban_key)     --> nil

-- 使用 rawset 绕过元方法，为被禁止键赋值
rawset(secure_table, "ban_key", "new")
print(secure_table.ban_key)     --> new
```

### 练习 1：实现向量模块

实现 `vector.lua` 向量模块：
1. 支持加法、减法、乘法、除法
2. 支持负号操作
3. 支持字符串表示

**示例**：
```lua
local Vector = require("vector")

local v1 = Vector.new(3, 4)
local v2 = Vector.new(1, 2)

print(v1)           --> Vector(3, 4)
print(v2)           --> Vector(1, 2)
print(v1 + v2)      --> Vector(4, 6)
print(v1 - v2)      --> Vector(2, 2)
print(v1 * v2)      --> Vector(3, 8)
print(v1 / v2)      --> Vector(3, 2)
print(-v1)          --> Vector(-3, -4)
```
